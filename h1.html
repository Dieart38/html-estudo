Tags h1 a h6

As tags <h1> a <h6> em HTML são usadas para definir títulos e subtítulos em uma página web. Elas ajudam a organizar o conteúdo de forma hierárquica, do mais importante ao menos importante. Aqui está uma explicação simples:

<h1>: É o título principal da página. Deve ser usado apenas uma vez por página para indicar o tema principal.

<h2>: É um subtítulo, usado para seções principais dentro do conteúdo. Pode haver vários <h2> em uma página.

<h3>: É um subtítulo de nível inferior, usado para subseções dentro de uma seção <h2>.

<h4>: É um subtítulo ainda mais específico, usado dentro de uma seção <h3>.

<h5>: É um subtítulo de nível inferior, usado dentro de uma seção <h4>.

<h6>: É o subtítulo de menor importância, usado dentro de uma seção <h5>.

Aqui está um exemplo de como essas tags podem ser usadas:

                <!DOCTYPE html>
                <html>
                <head>
                    <title>Exemplo de Títulos</title>
                </head>
                <body>
                    <h1>Título Principal</h1>
                    <h2>Seção Principal</h2>
                    <h3>Subseção</h3>
                    <h4>Subseção de Nível 4</h4>
                    <h5>Subseção de Nível 5</h5>
                    <h6>Subseção de Nível 6</h6>
                </body>
                </html>
