Tag img

A tag <img> é usada em HTML para incorporar imagens em uma página web. Aqui estão alguns pontos importantes sobre ela:

Sintaxe Básica:

    <img src="caminho/para/imagem.jpg" alt="Descrição da imagem">

src: Especifica o caminho para a imagem.

alt: Fornece uma descrição alternativa da imagem, útil para acessibilidade e quando a imagem não pode ser carregada.

Atributos Comuns:

  • width e height: Definem a largura e a altura da imagem.
  • title: Adiciona um texto que aparece quando o usuário passa o mouse sobre a imagem.
  • loading: Pode ser usado para carregar imagens de forma preguiçosa (lazy), melhorando o desempenho da página.

Exemplo Completo:

    <img src="exemplo.jpg" alt="Exemplo de imagem" width="500" height="300" title="Imagem de Exemplo" loading="lazy">

A tag <img> é essencial para adicionar elementos visuais às páginas web, tornando-as mais atraentes e informativas.
