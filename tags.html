O que são TAGS

Tags

Tags em HTML são elementos fundamentais usados para estruturar e formatar o conteúdo de uma página web. Elas são instruções que informam ao navegador como exibir o conteúdo. Cada tag é delimitada por colchetes angulares < > e geralmente vem em pares: uma tag de abertura e uma tag de fechamento. Por exemplo, a tag de parágrafo é escrita como <p> para abrir e </p> para fechar.

Exemplo: <p> Curso de html </p>

Aqui estão alguns exemplos de tags comuns em HTML:

  • <h1> a <h6>: Usadas para definir cabeçalhos, com <h1> sendo o mais importante e <h6> o menos importante.
  • <p>: Define um parágrafo.
  • <a>: Cria um link (âncora).
  • <img>: Insere uma imagem.
  • <ul> e <li>: Criam listas não ordenadas e itens de lista, respectivamente.
  • <div>: Define uma divisão ou seção em um documento.

Uma página HTML básica começa com a tag <html>, seguida por <head> (onde você coloca informações sobre a página) e <body> (onde vai o conteúdo visível). Aqui está um exemplo:

                    <!DOCTYPE html>
                    <html>
                    <head>
                        <title>Minha Página Web</title>
                    </head>
                    <body>
                        <h1>Bem-vindo!</h1>
                        <p>Este é um parágrafo de exemplo.</p>
                    </body>
                    </html>
