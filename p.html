Tag P

A tag <p> em HTML é usada para definir um parágrafo de texto. Aqui estão alguns pontos importantes sobre ela:

Estrutura: A tag <p> é um elemento de bloco, o que significa que ela ocupa toda a largura disponível e começa em uma nova linha.

Espaçamento: Navegadores geralmente adicionam uma linha em branco antes e depois de cada parágrafo para separá-los visualmente.

Uso: É usada para agrupar conteúdos relacionados, como texto, imagens ou campos de formulário.

Aqui está um exemplo simples de como usar a tag <p>:

                <p>Este é um parágrafo de exemplo.</p>
